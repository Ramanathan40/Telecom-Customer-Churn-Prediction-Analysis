# Telecom-Customer-Churn-Prediction-Analysis
Machine Learning Project on Telecom Customer Churn Prediction using SQL, Python and Power BI

Data Analytics and Machine Learning Pipeline with SQL Server, Python and Power BI

Project Overview

- This project demonstrates an end-to-end data analytics workflow that integrates Microsoft SQL Server, Python and Power BI to transform raw data into actionable business insights and predictive analytics.

- The workflow begins with loading a dataset into Microsoft SQL Server, followed by extensive data exploration and cleaning. After preprocessing the data, a Random Forest Machine Learning model is developed to generate predictions on new data. Finally, the processed data and analytical findings are visualized through an interactive Power BI dashboard.

- The project showcases essential data analyst and data science skills, including database management, data cleaning, exploratory data analysis (EDA), machine learning and business intelligence reporting.

Objectives
- Load and manage datasets using Microsoft SQL Server.
- Perform data exploration and quality assessment.
- Identify and handle missing (NULL) values.
- Create cleaned datasets and store them in new SQL tables.
- Perform data preprocessing for machine learning.
- Build and evaluate a Random Forest predictive model.
- Generate predictions on new/unseen data.
- Develop interactive dashboards using Power BI.
- Deliver actionable insights through data visualization.

This project demonstrates a complete data analytics lifecycle, from database management and data cleaning to machine learning and business intelligence reporting.
